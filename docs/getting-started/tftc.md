# tftc.top 使用指南 { #tftc-guide }

[tftc.top](https://tftc.top/) 是一个由热心宝友开发的第三方寻宝地图网站，提供了一个简洁易用的界面，帮助国内玩家在手机上查看附近的藏宝点信息。

默认情况下，这个页面刚打开只会显示最近 10 天内新增的宝。

## 如何设置语言为中文 { #set-language }

在页面左上角点击三条线的按钮，然后在下面寻找设置一栏，将语言设置成中文即可。

![修改语言](./imgs/tftc-1.jpg){ width="50%" }

## 如何查看附近的藏宝点 { #view-caches }

打开页面左上角这两个按钮，然后将地图放大到你所在的位置附近，就可以看到附近的藏宝点了。点击藏宝点图标就可以查看该藏宝点的详细信息了。

![查看藏宝点](./imgs/tftc-2.jpg){ width="50%" }
