# c:geo 配置

## c:geo 如何登录？

在下载完软件后，打开 c:geo，将界面下方的导航栏切换到 `首页 (Home)`。点击右上角的三个点，并在新打开的下拉菜单中选择 `设置 (Settings)`。

![打开设置](./imgs/cgeo-1.jpg){ width="40%" }

在设置中依次点击 `服务 (Services)` → `Geocaching.com` → `激活 (Activate)` 进入登录界面，输入你的 Geocaching 账号和密码后点击登录即可。

![打开设置](./imgs/cgeo-2.jpg){ width="40%" }

![打开设置](./imgs/cgeo-3.jpg){ width="40%" }

![打开设置](./imgs/cgeo-4.jpg){ width="40%" }

当你在 c:geo 的首页中能看到类似「登录成功」的字样时，就说明你已经成功登录了。

![打开设置](./imgs/cgeo-5.jpg){ width="40%" }

## 使用 c:geo 可能遇到的各种问题

???+ info "我注册时是通过第三方渠道登录的，但 c:geo 只能通过账号和密码登录，怎么办？"
    这是因为 c:geo 目前只支持账号密码登录方式，而不支持第三方登录（如 Google、Apple 等）。

    如果你注册时是通过第三方渠道登录的，则需要退出登录后，通过官网的 [忘记密码](https://www.geocaching.com/account/signin/requestpassword) 功能，设置一个新的密码。
    （注：设置密码并不影响你之后继续通过第三方渠道登录）

    密码设置完成后，你就可以用新的密码登录 c:geo 了。

???+ info "显示登录失败，且出错信息中包含 because of wrong username/password"
    首先排除输入错误的可能性，确认你输入的账号和密码是正确的。

    如果确认无误但仍然登录失败，可能是因为 c:geo 的版本过旧，无法兼容当前的 Geocaching 登录接口。

    你可以尝试更新 c:geo 到 [最新版本](https://g.nano.ac/https://github.com/cgeo/cgeo/releases/download/market_20260102/cgeo-release_20260102.apk)。
